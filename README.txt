To execute the Python files, kindly create a python environment as described below.

Environment:
Python Version - 3.7.2

Python Packages
Numpy Version - 1.20.1 - pip install numpy
Pandas Version - 1.2.3 - pip install pandas
Matplotlib Version - 3.3.4 - pip install matplotlib
Keras Version - 2.4.3 - pip install Keras
Sklearn Version - 0.24.1 - pip install scikit-learn
seaborn version - 0.11.1 - pip install seaborn


Download the python(IDLE) from the link below. 
https://www.python.org/ftp/python/3.7.2/python-3.7.2.exe

Run IDLE 
Open the Hepatitis.py, Heart.py and execute them.
